# 1RM Calculator - IRON PROTOCOL™ Companion Tool

## 🏋️ Calculate Your One-Rep Maximum

A standalone Progressive Web App calculator using 7 different formulas to estimate your one-rep maximum.

### 📊 7 Calculation Formulas

1. **Epley (1985)** - Classic, widely used
2. **Brzycki (1993)** - Accurate for 1-10 reps
3. **Lombardi (1989)** - Good for powerlifters
4. **Mayhew (1992)** - Highly accurate
5. **Landers (1985)** - Best for bench press
6. **O'Conner (1989)** - Conservative estimate
7. **Wathan (1994)** - Excellent for squats

### 📚 Formulas Explained

**Epley:**
```
1RM = weight × (1 + reps/30)
```

**Brzycki:**
```
1RM = weight × 36/(37 - reps)
```

**Lombardi:**
```
1RM = weight × reps^0.10
```

**Mayhew:**
```
1RM = 100 × weight / (52.2 + 41.9 × e^(-0.055 × reps))
```

**Landers:**
```
1RM = weight × 100 / (101.3 - 2.67123 × reps)
```

**O'Conner:**
```
1RM = weight × (1 + reps/40)
```

**Wathan:**
```
1RM = 100 × weight / (48.8 + 53.8 × e^(-0.075 × reps))
```
