INTEGRATION INSTRUCTIONS FOR 1RM CALCULATOR
============================================

The 1RM Calculator (1RM-CALCULATOR.html) is a standalone tool that can be:

OPTION 1: STANDALONE USE
└─ Open 1RM-CALCULATOR.html directly in browser
└─ Works offline immediately
└─ Can be bookmarked on home screen
└─ Functions as separate PWA app

OPTION 2: EMBEDDED IN MAIN APP
└─ Add 1RM tab to iron-protocol/index.html
└─ Add link to main navigation
└─ Access from anywhere in app

OPTION 3: SIDE-BY-SIDE TOOLS
├─ Keep both apps separate
├─ Link from main app to 1RM
├─ Users can switch between tools
└─ Each loads independently

RECOMMENDATION: OPTION 1 (STANDALONE)
- Works perfectly as separate tool
- Minimal file overhead
- Users install as separate home screen app
- Both tools work offline

FILES INCLUDED:
- 1RM-CALCULATOR.html (11KB) - Complete standalone calculator
- All 7 formulas: Epley, Brzycki, Lombardi, Mayhew, Landers, O'Conner, Wathan

FEATURES:
✅ 7 calculation formulas
✅ Average & range calculation
✅ Summary statistics
✅ Formula explanation
✅ Offline support
✅ Mobile optimized
✅ PWA installable
✅ No dependencies

DEPLOYMENT:
1. Keep 1RM-CALCULATOR.html in project root (or /tools folder)
2. Share URL: https://username.github.io/repo-name/1RM-CALCULATOR.html
3. Users install as app to home screen
4. Works offline completely
